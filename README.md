# Описание в разработке
Проект математического моделирования гидродинамики жидкого припоя в условиях низкой гравитации в рамках эксперимента Роскосмоса и NASA на МКС.
Проект завершён и успешно выведен на орбиту в 2021 году.  
Статья в Physics of Fluids: [https://doi.org/10.1063/5.0039718](https://doi.org/10.1063/5.0039718)
